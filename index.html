Види виделок. Розбираємось в деталях

До винаходу виделки люди на Заході користувались лише ножем та ложкою, тому більшість їжі їли руками. Перша виделка з'явилась ще у IX столітті. А повіноцінним столовим приладдям виделки стали у XVII столітті.

Зараз ми не уявляємо прийому їжі без виделки, а розмаїття їх видів вражає. Розбираємомось в деталях

Поширені види

Обідня виделка

Звичайна виделка з 4-ма довгими зубцями. Призначена для вживання гарячих других страв. Сервірують її зліва від тарілки, зубцями догори

Виделка для риби

Виделка коротша ніж обідня. Має також 4 зубці. В залежності від виду риби, може мати закруглені краї та щілину між середніми зубцями. Подається поряд з обідньою виделкою.

Виделка для закусок

Призначена для їжі холодних страв. Точна зменшена копія обідньої виделки. Сервірується зліва від виделки для риби

Десертна виделка

Призначена для такої їжі, як пироги. Має 3 зубці та незвичайний дизайн. Кладеться поряд з десертною тарілкою. Довжина виделки, не більша діаметру тарілки

Виделка для фруктів

Подається, якщо на столі є фрукти і деякі види ягід. Має 2 зубці. Якщо фрукти свіжі, то сервірується разом з ножем

Допоміжні виделки

Лежать разом зі стравою, для якої призначені. Бувають: для лимонів, омарів, мідій, шпротів, раків, устриць, оселедця
